═══════════════════════════════════════════════════════════════════

WHAT TO LOOK FOR
═══════════════════════════════════════════════════════════════════

IN BROWSER CONSOLE (F12 → Console tab):
────────────────────────────────────────
✓ "Sending registration request:" - Shows data being sent
✓ "Response status: 200" - Server responded successfully
✓ "Response text:" - Shows what server returned
✓ "Parsed response:" - Shows JSON data

ERROR MESSAGES:
───────────────
✗ "Failed to fetch" → Apache not running or wrong URL
✗ "JSON parse error" → PHP error in response (not valid JSON)
✗ "Response status: 404" → File not found
✗ "Response status: 500" → PHP error on server

COMMON SOLUTIONS
═══════════════════════════════════════════════════════════════════

SOLUTION 1: Restart Apache
───────────────────────────
1. Open XAMPP Control Panel
2. Stop Apache
3. Start Apache
4. Try again

SOLUTION 2: Clear Browser Cache
────────────────────────────────
1. Press Ctrl + Shift + Delete
2. Clear "Cached images and files"
3. Press Ctrl + F5 to hard refresh
4. Try again

SOLUTION 3: Check File Paths
─────────────────────────────
Ensure files are at:
C:\xampp\htdocs\smart-parking-system\

Required files:
✓ register.html
✓ api\auth.php
✓ config\database.php
✓ js\auth.js

SOLUTION 4: Check Apache Error Log
───────────────────────────────────
Open: C:\xampp\apache\logs\error.log
Look for recent PHP errors (check timestamp)
