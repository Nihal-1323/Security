SUPER SIMPLE 5-STEP SETUP
═══════════════════════════════════════════════════════════════════


STEP 1: Install XAMPP
──────────────────────────────────────────────────────────────────
1. Go to: https://www.apachefriends.org/
2. Click "Download" (Windows version)
3. Run the installer
4. Install to: C:\xampp (default location)
5. Finish installation


STEP 2: Download from GitHub
──────────────────────────────────────────────────────────────────
1. Go to: https://github.com/Nihal-1323/Security
2. Click the green "Code" button
3. Click "Download ZIP"
4. Save the ZIP file


STEP 3: Extract Files
──────────────────────────────────────────────────────────────────
1. Find the downloaded ZIP file
2. Right-click → "Extract All"
3. Extract it anywhere (like Desktop or Downloads)
4. Open the extracted folder
5. Find the "smart-parking-system" folder inside
6. Copy this folder
7. Paste it into: C:\xampp\htdocs\

Final location should be:
→ C:\xampp\htdocs\smart-parking-system\


STEP 4: Start XAMPP
──────────────────────────────────────────────────────────────────
1. Search "XAMPP" in Windows Start menu
2. Open "XAMPP Control Panel"
3. Click "Start" next to Apache (wait for it to turn GREEN)
4. Click "Start" next to MySQL (wait for it to turn GREEN)

Both should be GREEN now!


STEP 5: Run Setup
──────────────────────────────────────────────────────────────────
1. Open folder: C:\xampp\htdocs\smart-parking-system\
2. Find file: setup.bat
3. Double-click it
4. Wait for "Setup completed successfully!"
5. Press any key to close

This will:
✓ Create the database
✓ Create all tables
✓ Add 14 parking slots (A1-D3)
✓ Add admin user


STEP 6: Open Website
──────────────────────────────────────────────────────────────────
1. Open your web browser (Chrome, Firefox, Edge)
2. Type: http://localhost/smart-parking-system/
3. Press Enter

You should see the Smart Parking System! 🎉

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

PROBLEM: Website not loading
─────────────────────────────
✓ Check XAMPP Control Panel
✓ Apache should be GREEN
✓ If not, click "Start" for Apache


PROBLEM: Connection error
──────────────────────────
✓ Check XAMPP Control Panel
✓ MySQL should be GREEN
✓ If not, click "Start" for MySQL
✓ Run setup.bat again


PROBLEM: No parking slots showing
──────────────────────────────────
✓ Run setup.bat again
✓ This will add the 14 parking slots


PROBLEM: MySQL won't start (stays red)
───────────────────────────────────────
1. Go to: C:\xampp\htdocs\smart-parking-system\
2. Double-click: diagnose-mysql.bat
3. Follow the instructions
4. Or double-click: fix-mysql.bat
