booking, payments, and admin features.


═══════════════════════════════════════════════════════════════════

SUPER QUICK START (5 MINUTES)
═══════════════════════════════════════════════════════════════════

1. Install XAMPP
   Download: https://www.apachefriends.org/
   Install to: C:\xampp

2. Extract this ZIP
   Copy "smart-parking-system" folder to: C:\xampp\htdocs\

3. Start XAMPP
   Open XAMPP Control Panel
   Start Apache (click Start button)
   Start MySQL (click Start button)

4. Run Setup
   Go to: C:\xampp\htdocs\smart-parking-system\
   Double-click: setup.bat
   Wait for completion

5. Open Website
   Browser: http://localhost/smart-parking-system/

Done! 🎉
