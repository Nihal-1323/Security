REQUIREMENTS

✓ Windows PC
✓ XAMPP (free download from: https://www.apachefriends.org/)


INSTALLATION STEPS
═══════════════════════════════════════════════════════════════════

STEP 1: Install XAMPP
──────────────────────
1. Download XAMPP from: https://www.apachefriends.org/
2. Run the installer
3. Install to default location: C:\xampp
4. Complete the installation


STEP 2: Extract the ZIP file
─────────────────────────────
1. Extract the ZIP file you received
2. Copy the "smart-parking-system" folder
3. Paste it into: C:\xampp\htdocs\
4. Final path should be: C:\xampp\htdocs\smart-parking-system\


STEP 3: Start XAMPP Services
─────────────────────────────
1. Open XAMPP Control Panel (search in Windows Start menu)
2. Click "Start" for Apache
3. Click "Start" for MySQL
4. Both should turn GREEN


STEP 4: Import Database (AUTOMATIC)
────────────────────────────────────
1. Go to: C:\xampp\htdocs\smart-parking-system\
2. Double-click: setup.bat
3. Wait for it to complete
4. Press any key to close

OR (if setup.bat doesn't work):

1. Go to: C:\xampp\htdocs\smart-parking-system\
2. Double-click: import-database.bat
3. Wait for "Database imported successfully!"
4. Press any key to close


STEP 5: Open the Website
─────────────────────────
Open your browser and go to:
→ http://localhost/smart-parking-system/

You should see the Smart Parking System homepage!

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

PROBLEM: "Site can't be reached"
─────────────────────────────────
✓ Check XAMPP Control Panel - Apache should be GREEN
✓ If not green, click "Start" for Apache
✓ Try again: http://localhost/smart-parking-system/


PROBLEM: "Connection error" when registering
─────────────────────────────────────────────
✓ Check XAMPP Control Panel - MySQL should be GREEN
✓ If not green, click "Start" for MySQL
✓ Run: import-database.bat (in smart-parking-system folder)
✓ Try again


PROBLEM: "Table doesn't exist" error
─────────────────────────────────────
✓ Database not imported yet
✓ Run: import-database.bat
✓ Wait for "Database imported successfully!"
✓ Try again


PROBLEM: MySQL won't start (red in XAMPP)
──────────────────────────────────────────
✓ Run: diagnose-mysql.bat (in smart-parking-system folder)
✓ Follow the instructions shown
✓ OR run: fix-mysql.bat
