Choose the right script for your system:

  • setup.bat - No Python required (easiest)
  • setup.py - Python script (recommended)

  • check-mysql.py - Check MySQL status

═══════════════════════════════════════════════════════════════

🎯 INSTALLATION PROCESS
═══════════════════════════════════════════════════════════════

The setup script will:
  1. Find your XAMPP/WAMP installation
  2. Copy all files to web root
  3. Create MySQL database
  4. Import schema and default data
  5. Open test page in browser
  6. Show you the access URLs

Total time: ~5 minutes
